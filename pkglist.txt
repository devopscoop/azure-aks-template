# Arch Linux package list for azure-aks-template — the pacman/AUR equivalent
# of the Brewfile. Installs every CLI tool used or referenced by this repo.
#
# Requires an AUR helper (yay or paru) because tenv-bin is an AUR package;
# everything else is in the official repos.
#
# Usage:
#   grep -vE '^(#|$)' pkglist.txt | yay -S --needed -
#   grep -vE '^(#|$)' pkglist.txt | paru -S --needed -

# Azure CLI (`az`) — bootstrapping and `az aks get-credentials`
azure-cli

# tenv — AUR; installs/pins OpenTofu (`tofu`) from .opentofu-version, keeping
# local runs locked to the same version as CI. Do NOT install opentofu directly.
tenv-bin

# git — committing .terraform.lock.hcl changes and the GitOps workflow
git

# kubectl — using the cluster after `az aks get-credentials` writes the kubeconfig
kubectl
